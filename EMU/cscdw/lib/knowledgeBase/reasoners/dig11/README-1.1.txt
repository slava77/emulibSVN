# Synopsis #

XML schemas and Java implementation of the DIG interface -- 
a standardised XML interface to Description Logics <dl.kr.org>
systems developed by the DL Implementation Group 
(DIG <dl.kr.org/dig>). 
       
The implementation consists of:
a) an XMLBeans API automatically derived from the DIG 1.1 XML schema
using Apache Foundation's XMLBeans <xml.apache.org/xmlbeans>;
b) a Reasoners Java API using the above to actually communicate with
DIG reasoners.

# Download #

Schema and source available from the DIG Sourceforge website
<sourceforge.net/projects/dig>.

# Distribution #

This distribution contains:

dig-1.1.xsd          - the DIG 1.1 XML schema
dig1.1-xmlbeans.jar  - the DIG 1.1 XMLBeans
dig1.1-reasoners.jar - the Java DIG 1.1 reasoners API
javadoc/             - javadoc for the above
lib/                 - xmlbeans required libraries
